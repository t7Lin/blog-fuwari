#### **一、 名词后缀**

这些后缀附加在动词或形容词后，构成抽象或具体的名词。

|后缀|含义/功能|例词|
|---|---|---|
|**-ment**|表示行为、状态或结果|develop -> **development**, agree -> **agreement**, govern -> **government**|
|**-ion, -tion, -sion**|表示行为、过程或状态|act -> **action**, educate -> **education**, decide -> **decision**|
|**-ery, -ry**|表示状态、行为、场所或总称|brave -> **bravery**, slave -> **slavery**, bake -> **bakery**|
|**-ist**|表示“从事……的人”或“信仰……主义者”|art -> **artist**, science -> **scientist**, social -> **socialist**|
|**-ism**|表示“主义、学说、行为或特征”|social -> **socialism**, capital -> **capitalism**, critic -> **criticism**|
|**-th**|表示状态、性质或过程（常伴随音变）|warm -> **warmth**, long -> **length**, strong -> **strength**|
|**-ness**|加在形容词后，构成抽象名词|happy -> **happiness**, kind -> **kindness**, dark -> **darkness**|
|**-ship**|表示状态、身份、职位或关系|friend -> **friendship**, leader -> **leadership**, citizen -> **citizenship**|
|**-hood**|表示时期、状态或身份|child -> **childhood**, brother -> **brotherhood**, neighbor -> **neighborhood**|

#### **二、 形容词后缀**

这些后缀附加在名词或动词后，构成描述性的形容词。

|后缀|含义/功能|例词|
|---|---|---|
|**-ing**|表示“令人……的”（主动意义）|interest -> **interesting**, surprise -> **surprising**, bore -> **boring**|
|**-ed**|表示“感到……的”（被动意义）|interest -> **interested**, surprise -> **surprised**, bore -> **bored**|
|**-al**|表示“与……有关的”|nature -> **natural**, education -> **educational**, music -> **musical**|
|**-tic, -ic**|表示“属于……的、具有……性质的”|romance -> **romantic**, hero -> **heroic**, energy -> **energetic**|
|**-ly**|加在名词后，表示“具有……特征的”|friend -> **friendly**, year -> **yearly**, order -> **orderly**|
|**-able, -ible**|表示“能够……的、值得……的”|accept -> **acceptable**, read -> **readable**, sense -> **sensible**|
|**-ious, -ous**|表示“充满……的、有……性质的”|danger -> **dangerous**, fame -> **famous**, ambition -> **ambitious**|
|**-ful**|表示“充满……的、有……倾向的”|beauty -> **beautiful**, care -> **careful**, wonder -> **wonderful**|
|**-less**|表示“没有……的”|hope -> **hopeless**, home -> **homeless**, care -> **careless**|
|**-y**|表示“充满……的、像……的”|cloud -> **cloudy**, sun -> **sunny**, health -> **healthy**|

#### **三、 其他重要词缀**

除了后缀，前缀和动词后缀也同样重要。

**1. 常见前缀**  
前缀通常改变词义，而不改变词性。

|前缀|含义|例词|
|---|---|---|
|**un-, in-, im-, il-, ir-**|表示“不、非”|happy -> **unhappy**, correct -> **incorrect**, possible -> **impossible**|
|**re-**|表示“再次、重新”|write -> **rewrite**, build -> **rebuild**, use -> **reuse**|
|**dis-**|表示“否定、相反”|agree -> **disagree**, appear -> **disappear**, honest -> **dishonest**|
|**mis-**|表示“错误地”|understand -> **misunderstand**, behave -> **misbehave**, lead -> **mislead**|
|**pre-**|表示“在……之前”|view -> **preview**, pay -> **prepay**, war -> **prewar**|
|**sub-**|表示“在……之下、次要的”|way -> **subway**, title -> **subtitle**, marine -> **submarine**|

**2. 动词后缀**

|后缀|功能|例词|
|---|---|---|
|**-ize, -ise**|表示“使成为……、……化”|modern -> **modernize**, summary -> **summarize**|
|**-en**|表示“使变成、成为”|weak -> **weaken**, length -> **lengthen**|
|**-ify**|表示“使……化、使成为”|simple -> **simplify**, pure -> **purify**|

#### **四、 复合词**

由两个或两个以上的词组合而成，构成新词。

|类型|构成方式|例词|
|---|---|---|
|**复合名词**|名词+名词， 动词+名词等|**football**, **breakfast**, **swimming pool**|
|**复合形容词**|形容词+名词， 名词+分词等|**well-known**, **good-looking**, **part-time**|
|**带连字符的复合形容词**|常用于作定语|**a five-year-old boy**, **a last-minute decision**, **a user-friendly design**|
