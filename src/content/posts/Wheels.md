## Lead in
Have you ever bought a train ticket in English?
Today we're going to learn expressions and intonation in this situation.
## Presentation
Intonation is the rise and fall of our voice. It can change the meaning of a sentence.
Let's listen a dialogue between a passenger and a ticket clerk.
Mark out rising and falling
Let's check if you have identified correctly. 
```
|---------------------------------------------|
|         **Intonation: The Music of Meaning**        |
|                                             |
|  A: I'd like a ticket... (Neutral)          |
|  B: Single ↗ or return ↘?   (Choices)       |
|  A: Single ↗.             (Polite Confirm)  |
|  B: Smoking ↗ or non-smoking ↘? (Choices)   |
|  A: Non-smoking ↗, please. (Polite Confirm) |
|  A: What platform...? ↘    (Wh- Question)   |
|  A: What time...? ↘        (Wh- Question)   |
|                                             |
|  **↑ Rise: Polite, Choices, Unfinished**    |
|  **↓ Fall: Serious, Finished, Wh-Questions**|
|---------------------------------------------|
```
