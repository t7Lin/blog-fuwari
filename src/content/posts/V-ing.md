## V-ing form as gerund

| aspect            | function                                          | examples                                                                                                                                |
| ----------------- | ------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| attribute*        | modify n. ,to explain the usage or function of n. | a swimming pool;(the pool that be used for swimming)<br>a reading book(a book for reading)<br>building/catering/publishing industry<br> |
| predicative       | issue, to explain identity or comment of subject  | My hobby is listening to music.                                                                                                         |
| subject           | issue                                             | Swimming is good for your health.                                                                                                       |
| object            | issue                                             | We enjoy bathing in the sun ray.                                                                                                        |
| object complement | issue                                             | He spent time improving his spoken English.                                                                                             |
| prep. object      | issue                                             | We can keep health by doing exercises.<br>You'll be responsible for talking with visitors.                                              |
## V-ing as present participle

| aspect            | function                                                                       | example                                                                                       |
| ----------------- | ------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------- |
| attribute         | a happening action,status                                                      | a running man(a man that is running)                                                          |
| predicative       | adjective,to describe feature,effect of subject                                | The movie is interesting/disappointing<br>~~The baby is sleeping.(present continuous tense)~~ |
| adverbial         | to describe when,where,why,how about the n.(absolute phrase/nominate absolute) | Walking in the park,he saw a friend.(When he was walking...)                                  |
| object complement | a happening action,status                                                      | I saw you crossing the road.                                                                  |
